# Physics 110: Introductory Mechanics

A comprehensive course website for Physics 110 built with GitHub Pages.

## Course Structure
- **Lectures**: Video lectures, slides, and notes
- **Notebooks**: Interactive Jupyter notebooks
- **Homework**: Problem sets and solutions
- **Quizzes**: Weekly assessments
- **Resources**: Additional learning materials

## Technology Stack
- HTML5, CSS3, JavaScript
- Tailwind CSS for styling
- GitHub Pages for hosting
- Jupyter notebooks for interactive content

## Local Development
1. Clone the repository
2. Open `index.html` in a web browser
3. For full functionality, serve via local server (e.g., `python -m http.server 8000`)

## Deployment
The site is automatically deployed to GitHub Pages from the main branch.
