1.1 Unit Conversions

Introduction to Unit Conversions

In physics, we often need to convert between different units. The key is to use conversion factors that equal 1, so we don't change the value, just the units.

For example: 1 km = 1000 m, so 1 km/1000 m = 1 and 1000 m/1 km = 1.

Example 1: Converting Kilometers to Meters

Let's convert 15.6 km to meters using Python:

# Convert kilometers to meters distance_km = 15.6 distance_m = distance_km * 1000 # 1 km = 1000 m print(f"{distance_km} km = {distance_m} m")

Example 2: Converting mph to m/s

Convert 45 miles per hour to meters per second:

# Convert mph to m/s speed_mph = 45 speed_ms = speed_mph * 1609.34 / 3600 # 1 mile = 1609.34 m, 1 hour = 3600 s print(f"{speed_mph} mph = {speed_ms:.2f} m/s")

1.2 Significant Figures

Understanding Significant Figures

Significant figures represent the precision of a measurement. The rules are:

  • • All non-zero digits are significant
  • • Zeros between non-zero digits are significant
  • • Leading zeros are not significant
  • • Trailing zeros in a number with a decimal point are significant

Example: Counting Significant Figures

Let's write a Python function to count significant figures:

def count_significant_figures(number): """ Count the number of significant figures in a number. This is a simplified version that works for many cases. """ # Convert to string to analyze digits num_str = str(number) # Remove decimal point for counting if '.' in num_str: # For numbers with decimal point # Remove any negative sign if num_str[0] == '-': num_str = num_str[1:] # Split into integer and fractional parts parts = num_str.split('.') integer_part = parts[0] fractional_part = parts[1] # Count significant figures # All digits in integer part are significant (except leading zeros) # All digits in fractional part are significant # Remove leading zeros from integer part integer_part = integer_part.lstrip('0') # Count all digits sig_figs = len(integer_part) + len(fractional_part) return sig_figs else: # For numbers without decimal point # Remove any negative sign if num_str[0] == '-': num_str = num_str[1:] # Remove leading zeros num_str = num_str.lstrip('0') # Count all digits (trailing zeros may or may not be significant) # In this simplified version, we'll count them return len(num_str) # Test the function test_numbers = [123, 0.00123, 1.2300, 100, 100.0] for num in test_numbers: sig_figs = count_significant_figures(num) print(f"{num} has {sig_figs} significant figures")

Example: Calculating Area with Significant Figures

Calculate the area of a rectangle with length = 21.3 cm and width = 9.8 cm, considering significant figures:

# Calculate area with significant figures length = 21.3 # 3 significant figures width = 9.8 # 2 significant figures # Calculate area area = length * width # The result should have 2 significant figures (same as the measurement with fewer sig figs) # Round to 2 significant figures area_rounded = round(area, 1 - len(str(int(area)))) print(f"Length: {length} cm") print(f"Width: {width} cm") print(f"Area: {area} cm²") print(f"Area (with sig figs): {area_rounded} cm²")

1.3 Dimensional Analysis

Introduction to Dimensional Analysis

Dimensional analysis is a method to check the consistency of physical equations. The basic dimensions are:

  • • Length [L]
  • • Time [T]
  • • Mass [M]

For an equation to be dimensionally consistent, all terms must have the same dimensions.

Example: Checking Dimensional Consistency

Let's check if the equation v² = u² + 2as is dimensionally consistent:

# Define dimensions # [L] = length, [T] = time # Dimensions of each term: # v²: (velocity)² = (L/T)² = L²/T² # u²: (velocity)² = (L/T)² = L²/T² # 2as: (acceleration × distance) = (L/T² × L) = L²/T² # All terms have dimensions L²/T², so the equation is dimensionally consistent print("v² has dimensions: L²/T²") print("u² has dimensions: L²/T²") print("2as has dimensions: L²/T²") print("The equation v² = u² + 2as is dimensionally consistent!")

Example: Finding Dimensions of Unknowns

The centripetal force F depends on mass m, velocity v, and radius r: F ∝ mᵃ vᵇ rᶜ Find the values of a, b, and c using dimensional analysis:

# Dimensions: # Force F: [M L T⁻²] # Mass m: [M] # Velocity v: [L T⁻¹] # Radius r: [L] # The equation: F = k mᵃ vᵇ rᶜ (where k is a dimensionless constant) # Dimensions: [M L T⁻²] = [M]ᵃ [L T⁻¹]ᵇ [L]ᶜ # [M L T⁻²] = [Mᵃ Lᵇ⁺ᶜ T⁻ᵇ] # Equating exponents: # For M: 1 = a # For L: 1 = b + c # For T: -2 = -b # Solving: a = 1 b = 2 c = 1 - b # From L: 1 = b + c => c = 1 - b print(f"a = {a}, b = {b}, c = {c}") print("So the formula is: F ∝ m v² / r")

1.4 Vector Operations

Introduction to Vectors

Vectors have both magnitude and direction. We can represent them using components:

A = Aₓ i + Aᵧ j

Where i and j are unit vectors in the x and y directions.

Example: Vector Operations in Python

Let's create a Vector class to perform vector operations:

class Vector: def __init__(self, x, y): self.x = x self.y = y def __str__(self): return f"({self.x}, {self.y})" def magnitude(self): return (self.x**2 + self.y**2)**0.5 def direction(self): # Returns angle in degrees from positive x-axis import math angle_rad = math.atan2(self.y, self.x) return math.degrees(angle_rad) def __add__(self, other): return Vector(self.x + other.x, self.y + other.y) def __sub__(self, other): return Vector(self.x - other.x, self.y - other.y) def dot(self, other): return self.x * other.x + self.y * other.y def cross(self, other): # For 2D vectors, cross product gives a scalar (z-component) return self.x * other.y - self.y * other.x # Create vectors A = Vector(3, 4) B = Vector(-2, 6) print(f"Vector A = {A}") print(f"Vector B = {B}") print(f"Magnitude of A = {A.magnitude():.2f}") print(f"Direction of A = {A.direction():.2f}°") print(f"A + B = {A + B}") print(f"A - B = {A - B}") print(f"A · B = {A.dot(B)}") print(f"A × B = {A.cross(B)}")

Example: Vector Addition Visualization

Let's visualize vector addition:

# We'll create a simple text-based visualization def visualize_vectors(A, B, result): print("Vector Visualization:") print("====================") print(f"A = ({A.x}, {A.y})") print(f"B = ({B.x}, {B.y})") print(f"A + B = ({result.x}, {result.y})") print() print("Graphical representation (simplified):") print(" y") print(" ↑") print(" |") # Find the maximum y-value to scale our visualization max_y = max(A.y, B.y, result.y, 0) # Create a simple text-based graph for y in range(int(max_y), -1, -1): line = f"{y:2d} |" for x in range(0, int(max(A.x, B.x, result.x, 0)) + 1): if A.x == x and A.y == y: line += " A" elif B.x == x and B.y == y: line += " B" elif result.x == x and result.y == y: line += " R" else: line += " ." print(line) print(" " + "-" * (2 * (int(max(A.x, B.x, result.x, 0)) + 1))) print(" " + " ".join(str(i) for i in range(0, int(max(A.x, B.x, result.x, 0)) + 1)) + " → x") # Create vectors A = Vector(3, 4) B = Vector(2, 2) result = A + B visualize_vectors(A, B, result)

1.5 Physical Applications

Applying Physics Concepts

Now let's apply what we've learned to solve real physics problems.

Example: Centripetal Force Calculation

A particle moves in a circle of radius 5 m with speed 10 m/s. If its mass is 2 kg, calculate the centripetal force:

# Centripetal force calculation def centripetal_force(mass, velocity, radius): """ Calculate centripetal force: F = m v² / r """ return mass * velocity**2 / radius # Given values mass = 2 # kg velocity = 10 # m/s radius = 5 # m # Calculate centripetal force force = centripetal_force(mass, velocity, radius) print(f"Mass: {mass} kg") print(f"Velocity: {velocity} m/s") print(f"Radius: {radius} m") print(f"Centripetal force: {force} N") print(f"Centripetal force in kN: {force/1000} kN")

Example: Relative Velocity

A ship moves east at 12 m/s. A passenger runs north at 5 m/s relative to the ship. What is the passenger's velocity relative to the sea?

# Relative velocity calculation import math # Velocities v_ship = 12 # m/s, east v_passenger_relative = 5 # m/s, north relative to ship # The passenger's velocity relative to sea is the vector sum # v_passenger = v_ship + v_passenger_relative # In component form: # v_ship = (12, 0) # v_passenger_relative = (0, 5) v_passenger_x = 12 v_passenger_y = 5 # Magnitude of resultant velocity v_passenger = math.sqrt(v_passenger_x**2 + v_passenger_y**2) # Direction (angle from east) angle = math.degrees(math.atan2(v_passenger_y, v_passenger_x)) print(f"Ship velocity: {v_ship} m/s east") print(f"Passenger velocity relative to ship: {v_passenger_relative} m/s north") print(f"Passenger velocity relative to sea: {v_passenger:.2f} m/s") print(f"Direction: {angle:.2f}° north of east")
